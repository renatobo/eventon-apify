## Improvements

- Restrict the public MCP schema manifest so the granular capability matrix (`custom_event_api_capabilities`) and per-capability RSVP flags (`rsvp_attendees_enabled`, `rsvp_counts_enabled`) are returned only to authenticated administrators; anonymous callers receive the coarse availability booleans only.
- Cap the events `slug` filter at 100 values and apply `sanitize_title` to every value in both the string and array forms to keep the `post_name__in` query bounded.

## Bug Fixes

- Remove the orphaned RSVP delta-sync post meta (`_eventon_apify_updated_at_gmt`) during plugin uninstall.
